# 手順書ナビ

マニュアル・手順書・ワークフローを、**誰でも見やすく作成・閲覧・共有**できるシステムです。
業務に初めて携わる方や新たに配属された方が、必要な手順にすぐ辿り着けることを目指しています。

## 特長

- **インストール不要** — `index.html` をブラウザで開くだけ。サーバー不要。
- **見やすい閲覧・検索** — カテゴリ絞り込み、本文も含むキーワード検索、長い手順には自動目次。
- **かんたん作成・編集** — 専門知識なしで、ブロック（見出し / 本文 / 手順ステップ / 注記 / チェック項目 / 画像 / コード）を組み合わせて作成。
- **手順ステップ** — ステップは自動で番号付け。チェック項目は閲覧時にチェック可能。
- **印刷 / PDF** — 整ったレイアウトで印刷・PDF保存（ブラウザの印刷機能）。
- **共有・バックアップ** — JSON で書き出し / 読み込み。チームでデータを受け渡しできます。

## 使い方

1. `index.html` をダブルクリックしてブラウザで開く。
2. 左の一覧から文書を選んで閲覧、または右上「＋ 新規作成」で作成。
3. 上部の「🔍 検索」「カテゴリ」で絞り込み。
4. 共有するときは「📤 書出」でJSONを保存し、相手は「📥 読込」で取り込む。

> 初回起動時は使い方ガイドとサンプル文書（接客 / システム操作）が入っています。

## データの保存先

文書データはお使いのブラウザの `localStorage` に保存されます。
別の端末・ブラウザへ移す場合や、複数人で共有する場合は JSON の書き出し / 読み込みを利用してください。

## 技術

依存ライブラリなしの単一HTML（HTML + CSS + バニラ JavaScript）。ビルド不要。
